## What this GitHub org contains

This is the **public toolbox** behind Forex Strategy Studio:

- **MQL5 projects generated by the app** (exported strategies / EAs)
- **MQL5 utilities** (helpers, templates, HTTP/JSON glue, risk utilities)
- **Python / Node.js utilities** (local web proxy, crawler utilities, automation tooling)
- **Forex AI agent + REST API**, plus **MQL5 bridge utilities** to call the agent from MetaTrader 5 EAs

If you use the Android app to build a strategy, this GitHub helps you **take it further**: integrate, extend, automate, and deploy cleanly.

## ✨ Core features (from the app)

- **Signals out of the box** (RSI overbought/oversold, EMA crossover, Stochastic trend detection, multi-indicator confirmation)
- **No-code strategy builder** using industry-standard indicators (EMA, RSI, MACD, Stochastic, more)
- **Advanced backtesting** on-device with historical performance data
- **Export real MQL5 code** and run it inside **MetaTrader 5**
- **Professional metrics & reports**: win rate, expectancy, drawdown, profit factor, more

- **Mobile-first**: builder + backtester right on your phone

## Repository map (typical layout)

> Repo names may vary as the org grows, but the structure stays consistent.

### MQL5
- `mql5-generated/` — app exports (ready-to-compile projects)
- `mql5-utils/` — reusable helpers & templates
- `mql5-agent-bridge/` — call a local/remote REST agent from MQL5

### Python / Node.js
- `signal-agent/` — Forex AI Agent (REST API)
- `local-proxy/` — local web proxy utilities
- `crawler-utils/` — data collection / crawling utilities

## Quickstarts

### 1) Build → backtest → export → run in MetaTrader 5
1. Build your strategy in the **Forex Strategy Studio** Android app  
2. Backtest and validate metrics  
3. Export to **MQL5**  
4. Compile & run in **MetaTrader 5**

➡️ Download the app: https://play.google.com/store/apps/details?id=com.visualstrategystudio

### 2) Run an AI agent → call it from an EA
1. Start the REST agent (Python/Node repo)
2. Use the MQL5 bridge utility to query it from your EA
3. Combine agent outputs with your rule-based strategy + risk controls
