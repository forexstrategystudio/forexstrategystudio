<h2>✨ Trading the way it was meant to be</h2>

- **Signals out of the box** (RSI overbought/oversold, EMA crossover, Stochastic trend detection, multi-indicator confirmation)
- **No-code strategy builder** using industry-standard indicators (EMA, RSI, MACD, Stochastic and more)
- **Advanced backtesting** on-device with historical Forex data
- **Export real MQL5 code** and run it inside **MetaTrader 5**
- **Professional metrics & reports**: win rate, expectancy, drawdown, profit factor and more
- **Mobile-first**: builder + backtester right on your phone

<h3>This is the public repository for Forex Strategy Studio, where we publish:</h3>

- MQL5 projects generated by the app (exported strategies / EAs)
- MQL5 utilities (helpers, templates, HTTP/JSON glue)
- Python/Node.js utilities (local web proxy, crawler utilities, automation tooling)
- Forex AI agent with REST API, plus MQL5 bridge utilities to call the agent from MetaTrader 5 EAs

Use the Forex Strategy Studio Android app to build strategies without coding, then use the resources in this GitHub to help you take it further: integrate, extend, automate, and deploy cleanly.
